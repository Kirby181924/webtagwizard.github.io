    HTML: HyperText Markup Language

    Un HTML c'est un des 3 composents principaux pour créer un page web. Les 3 composants principaux sont: HTML, CSS et Javascript.

    Son objectif principal est definir la structure de votre contenu. Il se compose d'une série d'éléments que vous utiliserez pour enfermer différents éléments de contenu afin qu'ils aient l'air ou se comportent d'une certaine manière. Les balises englobantes peuvent faire d'un mot ou d'une image un lien hypertexte vers un autre site, mettre des mots en italique, agrandir ou réduire les polices, etc.
